Process an affiliate commission via Affiliates Manager after a sell digital downloads checkout

== Description ==

This addon allows you to integrate Sell Digital Downloads with the Affiliates Manager plugin.

When a user pay for a download on your site, this addon will check to see if the user was referred to your site by an affiliate. It will then give commission to the affiliate who referred the user.

This addon requires the [Affiliates Manager](https://wordpress.org/plugins/affiliates-manager/) and [Sell Digital Downloads](https://wordpress.org/plugins/sell-digital-downloads/) plugins.

Read the usage instructions [here](https://wpaffiliatemanager.com/affiliates-manager-sell-digital-downloads-integration/)

== Installation ==

1. Upload `affiliatemgr-sdd-integration` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
